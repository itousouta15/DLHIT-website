## DLHIT-Website
### 技術棧
- Next.js 13 (App Router)
- Tailwind CSS
- TypeScript
- ESLint

### 開發啟動
```bash
npm run dev
```

### 品質檢查與建置
```bash
npm run lint  
npm run build  
npm start      
```

### 結構
- app/: 使用 App Router，各頁面已預留段落與占位圖示
- components/: 站點框架、主題切換、動畫載入
- public/: 可放置logo與專題圖片

### 部署
部署至 Cloudflare Page
